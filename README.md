## Bienvenido

---

Es por medio de este repo que busco condensar de forma práctica toda la teoria y ejercios necesarios para consolidar las bases de programación algorítmica, explicando los fundamentos matemáticos de forma intuitiva y solventando de la información que permita desarrollar la lógica de programación necesaria.

Los ejercios estarán hechos en python y c# y toda la teoría estará disponible en formato PDF. Este es un proyecto personal y ya que no soy un experto, agradecería las sugerencias, corrrecciones y recomendaciones que puedas aportar.
